Frequently Asked Questions

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to manipulate the content and structure of HTML and XML documents dynamically.

You can select an element from the DOM using various methods such as:

  • document.getElementById('id') for selecting an element by its ID.
  • document.getElementsByClassName('className') for selecting elements by class name.
  • document.querySelector('selector') for selecting the first element that matches a specified CSS selector.
  • document.querySelectorAll('selector') for selecting all elements that match a specified CSS selector.

Event delegation is a technique where a single event listener is added to a parent element instead of individual listeners for each child element. This approach improves performance and allows for dynamic elements to be handled automatically.

You can manipulate an element's attributes and styles using JavaScript:

  • To change an attribute, use element.setAttribute('attribute', 'value').
  • To change styles, use element.style.property = 'value'.
